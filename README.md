# python-password-generator
📌 Description
This Random Password Generator is a Python project that creates secure passwords based on user-defined length. It uses the "random" and "string" modules to generate a mix of uppercase letters, lowercase letters, and numbers. The project demonstrates module usage, loops, string handling, and user input while producing strong, random passwords.

✨ Features

User-defined password length
Ensures a minimum length of 2 characters
Includes at least one letter and one digit
Randomly shuffles characters for better security
Easy-to-understand Python implementation

🛠️ Requirements
Python 3.x

📚 Libraries Used
random
string


▶️ How to Run

1. Clone or download the project.
2. Open the project folder in a terminal.
3. Run the script:
python password_generator.py
4. Enter the required password length.
5. The program will display a randomly generated password.

💻 Sample Output

Enter password length: 8
Generated Password: a7K2mP9x
